# Treasure hunter game 


You can run game by following the steps below:

In the project directory:

### `Step 1`

```npm install && npm run install```


### `Step 2`


```npm run start ```
<br />
 Game is available at [http://localhost:3000](http://localhost:3000)


 ## Tests

 As a testing solution, I propouse using `@testing-library/react` library with `jest` framework. It allows to check if correct components and expected content is rendered. 
 You can find example tests and running instructions in the frontend project.
